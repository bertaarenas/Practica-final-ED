Este programa es un menú interactivo creado con java que se ejecuta en cosaola y utiliza la clase Scanner para leer los datos introducios por teclado. 
permite al usuario elegir entre varias opciones de utilidad. Cada opción ejecuta una pequeña función diferente que son :

1. Calculadora: realiza operaciones matemáticas básicas (+, −, ×, ÷).
2. Calculador de edad: calcula la edad a partir del año de nacimiento.
3. Conversor de temperatura: convierte grados Celsius a Fahrenheit y Kelvin.
4. Contador de palabras y caracteres: cuenta cuántas palabras y caracteres tiene una frase.
5. Generador de saludos: pide el nombre del usuario y muestra un saludo personalizado.
   
Después de ejecutar cualquier opción, el programa pregunta al usuario si desea volver al menú o salir.
